Contact

Get in touch at jvoigts@mit.edu.

Contact us if you have suggestions or problems with the design, so we can keep a list of open issues in order to improve later revisions.

If you're interested in helping to improve the design or methods described here, feel free to explore the github repository and create new branches of the source files.

If your protocol for drive fabrication deviates from the one described here, please send us a brief description of your methods so we can expand the protocol with variants that better suit different applications of the drive.


If you're using the design, please cite our paper:

Voigts J, Siegle JH, Pritchett DL and Moore CI (2013).
The flexDrive: An ultra-light implant for optical control and highly parallel chronic recording of neuronal ensembles in freely moving mice. Front. Syst. Neurosci. 7:8. doi: 10.3389/fnsys.2013.00008
