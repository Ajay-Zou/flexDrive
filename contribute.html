Contribute your designs and insights

If you're using the flexDrive design or a similar implant, it is likely that you have discovered procedures that could be helpful to others.

If so, please get in touch so we can add your insights to the protocol.

Similarly, if you are using other sources for the materials or tools, especially if you're outside the US, please let us know so we can include your sources. This also concerns the materials and tools that are not specified in metric units, such drills and tubing.

If you're interested in helping to improve the design or methods described here, or have ported the source files to other file formats, have a look at the github repository and create new branches of the source files and upload your designs.
